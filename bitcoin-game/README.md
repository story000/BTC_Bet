# Bitcoin Prediction Game

A real-time Bitcoin price prediction game built with React, Vite, and Tailwind CSS.

## Features

- Real-time price simulation
- Rise/Fall prediction mechanism
- Interactive betting system
- Live price chart visualization
- Responsive UI with animated effects

## Quick Start

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The game will open automatically at http://localhost:3000/

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## How to Play

1. Set your bet amount using the +/- buttons
2. Choose your prediction: RISE or FALL
3. Click "START GAME" to begin
4. Watch the 10-second countdown and live price chart
5. Win or lose based on whether the price moved in your predicted direction

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Lucide React (icons)
